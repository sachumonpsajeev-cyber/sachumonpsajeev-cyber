## 🧬 Thesis Project — Cardiac Arrest Prediction
**Multi-Window Intraoperative Cardiac Arrest Prediction Using Temporal Attention Networks**

* **Dataset:** VitalDB — 6,388 perioperative surgical cases.
* **Target:** AUROC > 0.91 (Benchmarked against Lee et al. 2024).
* **Predictive Windows:** 30, 60, 120, and 240 minutes pre-event.
* **Model Architecture:** TAN-LSTM (Temporal Attention Network + LSTM).
* **Optimization:** Handled 90:1 class imbalance using SMOTE-ENN.
