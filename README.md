## 🔬 Thesis Research — Cardiac Arrest Prediction

> **Multi-Window Intraoperative Cardiac Arrest Prediction Using Temporal Attention Networks**
> *TSI University · Supervisor: Prof. Emmanuel Merchan · Submission: April 2026*

| Component | Detail |
|---|---|
| **Dataset** | VitalDB — 6,388 perioperative surgical cases |
| **Architecture** | Two-Stage: LSTM (Stage 1) + TAN — Temporal Attention Network (Stage 2) |
| **Predictive Windows** | 30, 60, 120, and 240 minutes pre-event |
| **Final Model** | TAN+LSTM Hybrid Ensemble (AUROC: **0.9180**) |
| **Benchmark** | NEWS2 Baseline AUROC: 0.6295 — **+0.2885 improvement** |
| **Imbalance Handling** | SMOTE-ENN inside CV folds — 90:1 class ratio, zero data leakage |
| **Key Finding** | Attention weights monotonically increase 30→240 min, confirming progressive pre-arrest signal |

**Why this matters:** Existing early warning scores (NEWS2) were designed for ward settings and perform poorly intraoperatively. This is the first multi-window TAN model trained entirely on intraoperative VitalDB data — designed specifically for the OR.
