## 🧠 Thesis Project — Cardiac Arrest Prediction

**Multi-Window Intraoperative Cardiac Arrest Prediction**  
**Using Temporal Attention Networks on VitalDB Clinical Time-Series Data**

- 📊 Dataset: VitalDB — 6,388 perioperative surgical cases
- 🎯 Target: AUROC > 0.91 (benchmark: Lee et al. 2024)
- 🏥 Predicting cardiac arrest 30, 60, 120, and 240 minutes before the event
- 🤖 Model: TAN-LSTM (Temporal Attention Network + LSTM)
- ⚔️ Baselines: LightGBM · LSTM · Logistic Regression
- ⚖️ Class imbalance: 90:1 → handled with SMOTE-ENN
