## рЯФђ MSc Thesis вАФ Intraoperative Cardiac Arrest Prediction

> **Multi-Window Cardiac Arrest Prediction Using Temporal Attention Networks on VitalDB Clinical Time-Series**

Developed a two-stage deep learning pipeline to predict intraoperative cardiac arrest (CA) at 30, 60, 120, and 240 minutes before the event, using the publicly available VitalDB dataset (6,388 perioperative surgical cases).

| Component | Detail |
|---|---|
| **Dataset** | VitalDB вАФ 6,388 perioperative cases, Seoul National University Hospital |
| **Signals** | Heart rate, SpOвВВ, EtCOвВВ, MAP, SBP, DBP (36-dimensional feature vector) |
| **Architecture** | TAN-LSTM hybrid (4-head self-attention + LSTM ensemble) |
| **Best Result** | AUROC 0.9180 (95% CI: 0.8944вАУ0.9405) on multi-window sub-cohort |
| **Baselines** | LightGBM ¬Ј Random Forest ¬Ј XGBoost ¬Ј Logistic Regression ¬Ј LSTM |
| **Class Imbalance** | 1:26 to 1:96 вЖТ handled with Focal Loss, class-balanced weighting, Youden's J |
| **Validation** | 5-fold cross-validation, bootstrap DeLong test (p < 0.05) |
